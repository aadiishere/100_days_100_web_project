💰 Interactive Budget Tracker

Overview

Interactive Budget Tracker is a responsive web application that helps users manage their personal finances efficiently. Users can set a budget, track expenses, analyze spending patterns, and visualize category-wise expenses through an interactive pie chart.

The project is built using HTML, CSS, JavaScript, and Chart.js.

---

Features

Budget Management

- Set a total budget
- Track total spending
- View remaining balance

Expense Tracking

- Add new expenses
- Delete expenses
- Categorize expenses
- Real-time updates

Data Visualization

- Dynamic pie chart using Chart.js
- Category-wise expense analysis
- Spending distribution insights

User Experience

- Responsive design
- Dark mode support
- Clean and modern interface
- Local Storage persistence

How to Run

1. Clone the repository.

git clone <repository-url>

2. Navigate to the project folder.

cd Interactive-Budget-Tracker

3. Open "index.html" in your browser.

Alternatively, use a local server:

python -m http.server 8000

Visit:

http://localhost:8000

---

Usage

1. Set your budget.
2. Add expenses with category and amount.
3. Monitor spending statistics.
4. Analyze category-wise expenses through the pie chart.
5. Toggle dark mode for a better viewing experience.
6. Refresh the page without losing data thanks to Local Storage.
