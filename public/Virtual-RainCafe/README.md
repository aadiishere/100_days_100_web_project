## 🌧️ Overview

**Virtual Rain Cafe** is a modern productivity web application designed to recreate the relaxing atmosphere of working inside a cozy café during a rainy day.

Whether you're studying, coding, reading, writing, or simply relaxing, the application provides an immersive environment with animated rainfall, customizable ambient sounds, lo-fi music, focus timers, notes, and productivity tools.

Everything runs directly inside your browser with no backend required.

---

## ✨ Features

### 🎨 Beautiful Atmosphere Themes

Choose from multiple immersive environments:

* ☕ Cozy Coffeehouse
* 📚 Midnight Library
* 🌆 Neon Rain Alley
* 🌲 Forest Sanctuary

Each theme includes:

* Unique color palette
* Custom rain effects
* Different ambient atmosphere
* Dynamic accent colors

---

### 🌧️ Animated Rain Engine

A canvas-powered rain animation featuring:

* Adjustable rain intensity
* Drizzle
* Steady Rain
* Heavy Downpour
* Thunderstorm mode
* Dynamic splash effects
* Theme-specific rain physics
* Smooth animations using `requestAnimationFrame`

---

### 🎵 Ambient Sound Mixer

Create your perfect workspace by mixing sounds such as:

* 🌧️ Rain
* ⚡ Thunder
* 👥 Coffee Shop Ambience
* 🔥 Fireplace
* 🍃 Wind

Features include:

* Independent volume controls
* Individual mute buttons
* Global mute
* Automatic looping
* Persistent volume settings

---

### 🎧 Built-in Lo-Fi Music Player

Enjoy relaxing music while working.

Includes:

* Multiple lo-fi tracks
* Play/Pause controls
* Previous / Next track
* Interactive progress bar
* Seek functionality
* Volume control
* Animated vinyl record
* Music visualizer

---

### ⏳ Pomodoro Productivity Timer

Stay focused with an elegant timer.

Supports:

* Focus Session (25 min)
* Short Break
* Long Break

Features:

* Circular progress animation
* Pause / Resume
* Reset
* Completion sound
* Visual countdown

---

### ✅ Cafe Order Checklist

Organize your work using the built-in task manager.

Features:

* Add tasks
* Complete tasks
* Delete tasks
* Local storage persistence

---

### 📝 Cozy Notepad

A distraction-free writing space.

Features:

* Auto-save
* Character counter
* Download notes
* Clear notes
* Mechanical keyboard typing sounds

---

### 🌙 Focus Mode

Additional productivity tools include:

* Screen dimmer
* Hide UI mode
* Restore interface anytime
* Minimal distraction workspace

---

### 💾 Local Storage Support

Your preferences are automatically saved.

The application remembers:

* Selected theme
* Rain intensity
* Sound volumes
* Notes
* Tasks
* Music volume
* Keyboard sound preference
